    wxGraphicsMatrix (wx v2.5.2-rc0)

    View Source

    A wxGraphicsMatrix is a native representation of an affine matrix.

    The contents are specific and private to the respective renderer. Instances are ref counted and can therefore be assigned as usual. The only way to get a valid instance is via wxGraphicsContext:createMatrix/2 or wxGraphicsRenderer:createMatrix/2.

    This class is derived, and can use functions, from:

    • wxGraphicsObject
