Erlang User Conference

User conferences have been held annually in Stockholm since 1994 except 1996 when everybody was busy developing the first version of OTP (Open Telecom Platform). The photograph is from the Sixth EUC, October 3, 2000.

Proceedings from the following Erlang/OTP User Conferences are available on-line: 1999, 2000, 2001, 2002, 2003, 2004, 2005, 2006, 2007 and 2008.

The Erlang/OTP User Conference 2009 was arranged by Erlang Solutions.

See also information about Erlang workshops which are held in connection with the International Conference on Functional Programming (ICFP).
