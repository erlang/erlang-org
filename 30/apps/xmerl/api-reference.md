# xmerl v2.2-rc0 - API Reference

## Modules

- [xmerl](xmerl.md): Functions for exporting XML data to an external format.
- [xmerl_eventp](xmerl_eventp.md): Simple event-based processors (front-ends to `m:xmerl_scan`).
- [xmerl_sax_parser](xmerl_sax_parser.md): XML SAX parser API
- [xmerl_scan](xmerl_scan.md): Single pass XML scanner.
- [xmerl_xpath](xmerl_xpath.md): Xpath 1.0 search implementation.
- [xmerl_xs](xmerl_xs.md): XSLT-like XML document transformations.
- [xmerl_xsd](xmerl_xsd.md): Interface module for XML Schema validation.
