# observer v2.18.2 - API Reference

## Modules

- [crashdump_viewer](crashdump_viewer.md): A WxWidgets based tool for browsing Erlang crashdumps.
- [etop](etop.md): Erlang Top is a tool for presenting information about Erlang processes similar
to the information presented by "top" in UNIX.
- [observer](observer.md): A GUI tool for observing an Erlang system.
- [ttb](ttb.md): A base for building trace tools for distributed systems.
